🍈 The Melon Bread / めろんぱん

言葉のズレ、誤解、そして“結果が出た後の評価”をテーマにした、短編ブラウザゲームです。

企画者 / 開発者として3つの選択を行い、4つの結末へたどり着きます。

🇺🇸 English README → README.md

⸻

✨ 特徴

* 🌍 2つの世界観UI
    * 🇺🇸 Retro Future Edition（英語版）
    * 🇯🇵 ゆるUI版（日本語版）
* 🎮 4つのマルチエンディング
* 📋 SNSシェア文コピー機能
* 📱 レスポンシブ対応の単一HTML
* 🎨 言語ごとのデザイン切替

⸻

🎯 コンセプト

同じ誤解でも、結果によって呼ばれ方が変わります。

* 成功すれば 創造性
* 失敗すれば 確認不足

その構造を、遊べる体験として表現しました。

⸻

🚀 遊び方

1. 世界観を選ぶ（英語 / 日本語）
2. 役割を選ぶ
3. 3つの場面で選択する
4. 4つの結末のどれかに到達する
5. 結果をコピーしてシェアする

⸻

🧱 使用技術

* HTML
* CSS
* Vanilla JavaScript

ビルド不要、依存関係なし。ブラウザでそのまま遊べます。
